# averageコマンド
![test](https://github.com/Taichiichijo/robosys2024/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字の平均を出す。

## インストール方法

$git clone git@github.com:Taichiichijo/robosys2024.git

$cd tmp/robosys2024/

## 使い方

$python3 ./average.py

この後に数字を好き入力していき、最後にCtrl dで出力する
## 実行結果

平均値：nのように出る（nは数値が入る）

## 必要なソフトウェア
- Python
  - テスト済みバージョン：　3.7~3.11

## テスト環境
- Ubuntu 20.04
